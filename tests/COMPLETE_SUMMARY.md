## 🎓 Примеры использования

### Запуск конкретных категорий

```bash
# Только быстрые тесты
pytest -m "not slow"

# Только интеграционные
pytest -m integration

# Только UI тесты
pytest -m ui

# Только unit тесты
pytest -m unit
```

### Отладка

```bash
# Остановиться на первой ошибке
pytest -x

# Показать локальные переменные
pytest -l

# Запустить pdb при ошибке
pytest --pdb

# Показать print
pytest -s
```

### Отчеты

```bash
# HTML отчет о покрытии
pytest --cov=core --cov=infrastructure --cov-report=html

# XML отчет (для CI)
pytest --cov=core --cov-report=xml

# Терминал с пропущенными строками
pytest --cov=core --cov-report=term-missing
```
