## Запуск тестов

### Установка зависимостей
```bash
pip install -r tests/requirements-test.txt
```

### Запуск всех тестов
```bash
pytest
```

### Запуск с покрытием
```bash
pytest --cov=core --cov=infrastructure --cov=ui --cov-report=html
```

### Запуск по категориям
```bash
# Unit тесты
pytest -m unit

# Интеграционные тесты
pytest -m integration

# UI тесты
pytest -m ui

# Быстрые тесты (исключить медленные)
pytest -m "not slow"
```

### Параллельный запуск
```bash
pytest -n auto
```
