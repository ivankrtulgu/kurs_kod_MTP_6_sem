## Примечания

Все диаграммы представлены в текстовом формате ASCII для максимальной совместимости и простоты редактирования. Для создания визуальных диаграмм можно использовать инструменты:

- **PlantUML** — для UML диаграмм
- **Draw.io** — для блок-схем и диаграмм
- **Mermaid** — для диаграмм в Markdown
- **dbdiagram.io** — для ER-диаграмм

## Легенда

- **PK** — Primary Key (первичный ключ)
- **FK** — Foreign Key (внешний ключ)
- **UK** — Unique Key (уникальный ключ)
- **│** — связь один-к-одному
- ***** — связь один-ко-многим
- **→** — направление потока данных
- **↓** — последовательность действий
