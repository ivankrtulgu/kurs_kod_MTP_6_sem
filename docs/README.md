## Ключевые концепции

### Архитектура

Система построена на основе многослойной архитектуры:

```
┌─────────────────────────────────────┐
│     Presentation Layer (UI)         │
│   PyQt5 Windows & Widgets           │
└─────────────────────────────────────┘
              ↓
┌─────────────────────────────────────┐
│   Business Logic Layer (Core)       │
│   Services & Models                 │
└─────────────────────────────────────┘
              ↓
┌─────────────────────────────────────┐
│   Infrastructure Layer              │
│   Repositories & Database           │
└─────────────────────────────────────┘
```

### Основные компоненты

**Модели данных:**
- Book — книга в каталоге
- BookItem — инвентарный экземпляр
- Reader — читатель
- LoanRecord — запись о выдаче

**Сервисы:**
- BookService — управление каталогом
- InventoryService — управление инвентарем и выдачей
- QRService — генерация QR-кодов
- PrintingService — печать документов

**Репозитории:**
- BookRepository — работа с таблицей книг
- InventoryRepository — работа с инвентарем и читателями

## Соглашения

### Стиль кода

Проект следует PEP 8 с дополнениями:
- Максимальная длина строки: 100 символов
- Использование type hints
- Docstrings для всех публичных методов
- Комментарии на русском языке

### Именование

**Классы:** PascalCase
```python
class BookService:
    pass
```

**Функции и методы:** snake_case
```python
def add_book(self, book: Book) -> int:
    pass
```

**Константы:** UPPER_SNAKE_CASE
```python
DEFAULT_LOAN_DAYS = 14
```

**Приватные члены:** префикс _
```python
self._repository = repository
```

## Часто задаваемые вопросы

### Как добавить новую модель?

1. Создать файл в `core/models/`
2. Определить dataclass с полями
3. Добавить метод `validate()`
4. Реализовать `to_dict()` и `from_dict()`
5. Написать тесты

### Как добавить новый репозиторий?

1. Создать файл в `infrastructure/database/`
2. Реализовать методы CRUD
3. Добавить SQL-запросы
4. Написать тесты

### Как добавить новое окно?

1. Создать .ui файл в Qt Designer
2. Сгенерировать Python код: `pyuic5 file.ui -o ui_file.py`
3. Создать класс окна в `ui/windows/`
4. Подключить сигналы и слоты
5. Добавить в главное меню
