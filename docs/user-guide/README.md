## Введение

Система управления библиотекой предназначена для автоматизации основных процессов библиотечной деятельности:
- Ведение электронного каталога книг
- Учет инвентарных экземпляров
- Регистрация читателей
- Контроль выдачи и возврата литературы
- Генерация и печать QR-кодов

## Начало работы

### Запуск приложения

1. Запустите файл `main.py` или исполняемый файл приложения
2. Откроется главное окно системы с меню и панелью инструментов

### Главное окно

Главное окно состоит из:
- **Строка меню** — доступ ко всем функциям системы
- **Панель инструментов** — быстрый доступ к часто используемым операциям
- **Рабочая область** — MDI-область для открытия нескольких окон одновременно
- **Строка состояния** — отображение текущего количества книг в каталоге

### Структура меню

**Файл**
- Выход — закрыть приложение

**Каталог**
- Добавить книгу — добавление новой книги в каталог
- Список книг — просмотр всех книг
- Поиск — поиск книг по различным критериям
- Экспорт — экспорт каталога в CSV

**Инвентарь**
- Добавить экземпляры — создание инвентарных экземпляров
- Список экземпляров — просмотр всех экземпляров
- Выдать книгу — выдача книги читателю
- Вернуть книгу — возврат книги
- Активные выдачи — список всех выданных книг

**Читатели**
- Список читателей — управление читателями

**Сервис**
- OCR распознавание — распознавание библиографических данных

**Справка**
- О программе — информация о системе

## Управление каталогом

### Добавление книги

1. Выберите **Каталог → Добавить книгу** или нажмите кнопку на панели инструментов
2. Заполните обязательные поля:
   - **Автор** — фамилия и инициалы автора
   - **Название** — полное название книги
   - **ISBN** — международный стандартный книжный номер
3. Заполните дополнительные поля (опционально):
   - **Издательство** — название издательства
   - **Год издания** — год выпуска книги
   - **Количество страниц** — общее количество страниц
   - **Место издания** — город издания
   - **DOI** — цифровой идентификатор объекта
4. Добавьте обложку книги (опционально):
   - Нажмите кнопку **Выбрать файл**
   - Выберите изображение обложки
5. Нажмите **Сохранить**

**Примечания:**
- ISBN проверяется на корректность (поддерживаются форматы ISBN-10 и ISBN-13)
- Год издания должен быть в диапазоне 1900-2100
- Количество страниц должно быть положительным числом

### Просмотр списка книг

1. Выберите **Каталог → Список книг**
2. Откроется окно со списком всех книг в каталоге
3. В таблице отображаются:
   - ID книги
   - Автор
   - Название
   - ISBN
   - Издательство
   - Год издания

**Действия:**
- **Двойной клик** на книге — открыть карточку книги
- **Сортировка** — клик по заголовку столбца
- **Обновить** — обновить список

### Карточка книги

Карточка книги содержит:
- Полную информацию о книге
- Обложку книги (если добавлена)
- Библиографическую запись
- Список инвентарных экземпляров

**Доступные действия:**
- **Редактировать** — изменить данные книги
- **Удалить** — удалить книгу из каталога
- **Добавить экземпляры** — создать инвентарные экземпляры
- **Печать QR** — распечатать QR-коды

### Редактирование книги

1. Откройте карточку книги
2. Нажмите кнопку **Редактировать**
3. Измените необходимые поля
4. Нажмите **Сохранить**

### Удаление книги

1. Откройте карточку книги
2. Нажмите кнопку **Удалить**
3. Подтвердите удаление

**Внимание:** При удалении книги удаляются все её инвентарные экземпляры.

## Управление инвентарем

### Добавление инвентарных экземпляров

1. Выберите **Инвентарь → Добавить экземпляры**
2. Выберите книгу из списка
3. Укажите количество создаваемых экземпляров
4. Укажите местоположение (опционально)
5. Нажмите **Создать**

Система автоматически:
- Присвоит инвентарные номера
- Создаст QR-коды для каждого экземпляра
- Установит статус "Доступен"

### Просмотр инвентарных экземпляров

1. Выберите **Инвентарь → Список экземпляров**
2. Откроется список всех экземпляров с информацией:
   - Инвентарный номер
   - Название книги
   - Автор
   - Статус
   - Местоположение

### Статусы экземпляров

- **Доступен** — экземпляр доступен для выдачи
- **Выдан** — экземпляр выдан читателю
- **В ремонте** — экземпляр на реставрации
- **Утерян** — экземпляр утерян
- **Списан** — экземпляр списан

### Изменение статуса экземпляра

1. Откройте карточку экземпляра
2. Выберите новый статус из выпадающего списка
3. Нажмите **Сохранить**

**Примечание:** Нельзя изменить статус выданного экземпляра. Сначала необходимо оформить возврат.

### Изменение местоположения

1. Откройте карточку экземпляра
2. Введите новое местоположение
3. Нажмите **Сохранить**

## Работа с читателями

### Регистрация читателя

1. Выберите **Читатели → Список читателей**
2. Нажмите кнопку **Добавить читателя**
3. Заполните данные:
   - **Фамилия** (обязательно)
   - **Имя** (обязательно)
   - **Отчество** (опционально)
   - **Номер билета** (обязательно, уникальный)
   - **Телефон** (опционально)
   - **Email** (опционально)
4. Укажите срок действия читательского билета
5. Нажмите **Сохранить**

### Просмотр списка читателей

1. Выберите **Читатели → Список читателей**
2. В таблице отображаются:
   - ФИО читателя
   - Номер читательского билета
   - Телефон
   - Email
   - Дата регистрации
   - Срок действия билета
   - Статус (активен/заблокирован)

### Редактирование данных читателя

1. Откройте карточку читателя (двойной клик)
2. Измените необходимые данные
3. Нажмите **Сохранить**

### Блокировка читателя

1. Откройте карточку читателя
2. Установите флажок **Заблокирован**
3. Нажмите **Сохранить**

**Примечание:** Заблокированному читателю нельзя выдать книги.

### Удаление читателя

1. Откройте карточку читателя
2. Нажмите кнопку **Удалить**
3. Подтвердите удаление

**Внимание:** Нельзя удалить читателя с активными выдачами.

## Выдача и возврат книг

### Выдача книги

**Способ 1: Через меню**
1. Выберите **Инвентарь → Выдать книгу**
2. Выберите читателя из списка
3. Выберите книгу и экземпляр
4. Укажите срок выдачи (по умолчанию 14 дней)
5. Нажмите **Выдать**

**Способ 2: По QR-коду**
1. Выберите **Инвентарь → Выдать книгу**
2. Отсканируйте QR-код экземпляра
3. Выберите читателя
4. Нажмите **Выдать**

**Способ 3: По инвентарному номеру**
1. Выберите **Инвентарь → Выдать книгу**
2. Введите инвентарный номер
3. Выберите читателя
4. Нажмите **Выдать**

### Возврат книги

**Способ 1: Через меню**
1. Выберите **Инвентарь → Вернуть книгу**
2. Выберите экземпляр из списка выданных
3. Нажмите **Вернуть**

**Способ 2: По QR-коду**
1. Выберите **Инвентарь → Вернуть книгу**
2. Отсканируйте QR-код экземпляра
3. Нажмите **Вернуть**

**Способ 3: По инвентарному номеру**
1. Выберите **Инвентарь → Вернуть книгу**
2. Введите инвентарный номер
3. Нажмите **Вернуть**

### Просмотр активных выдач

1. Выберите **Инвентарь → Активные выдачи**
2. Откроется список всех выданных книг с информацией:
   - Читатель
   - Книга
   - Инвентарный номер
   - Дата выдачи
   - Дата возврата
   - Статус (в срок/просрочено)

**Фильтрация:**
- Все выдачи
- Только просроченные
- По конкретному читателю

### История выдач читателя

1. Откройте карточку читателя
2. Перейдите на вкладку **История выдач**
3. Просмотрите список всех выдач (текущих и прошлых)

## Печать QR-кодов

### Печать QR-кодов для книги

1. Откройте карточку книги
2. Нажмите кнопку **Печать QR**
3. Выберите режим печати:
   - **Только книга** — QR-код книги
   - **Книга + экземпляры** — QR-коды книги и всех экземпляров
   - **Только экземпляры** — QR-коды экземпляров
4. Укажите количество столбцов (1-4)
5. Нажмите **Создать PDF**
6. Выберите место сохранения файла

### Печать QR-кодов для экземпляров

1. Выберите **Инвентарь → Список экземпляров**
2. Выберите нужные экземпляры (Ctrl+клик для множественного выбора)
3. Нажмите кнопку **Печать QR**
4. Настройте параметры печати
5. Нажмите **Создать PDF**

### Формат QR-кодов

**QR-код книги содержит:**
- Тип: "book"
- ID книги
- ISBN
- Автор
- Название

**QR-код экземпляра содержит:**
- Тип: "item"
- ID экземпляра
- Инвентарный номер
- ID книги

## OCR-распознавание

Функция OCR позволяет автоматически распознать библиографические данные из изображения.

### Использование OCR

1. Выберите **Сервис → OCR распознавание**
2. Нажмите **Загрузить изображение**
3. Выберите изображение с библиографическими данными
4. Нажмите **Распознать**
5. Система автоматически извлечет:
   - Автора
   - Название
   - ISBN
   - Издательство
   - Год издания
   - Количество страниц
6. Проверьте распознанные данные
7. Нажмите **Добавить книгу** для сохранения в каталог

**Рекомендации:**
- Используйте четкие изображения
- Убедитесь, что текст хорошо читаем
- Избегайте бликов и теней
- Оптимальное разрешение: 300 DPI

## Поиск и фильтрация

### Поиск книг

1. Выберите **Каталог → Поиск**
2. Введите поисковый запрос
3. Поиск выполняется по полям:
   - Автор
   - Название
   - ISBN
4. Результаты отображаются в таблице

**Особенности поиска:**
- Поиск регистронезависимый
- Поддерживается частичное совпадение
- Можно искать по нескольким словам

### Фильтрация списков

В списках книг и экземпляров доступна фильтрация:
- По статусу (для экземпляров)
- По году издания (для книг)
- По издательству (для книг)

### Сортировка

Клик по заголовку столбца сортирует список:
- Первый клик — сортировка по возрастанию
- Второй клик — сортировка по убыванию

## Экспорт данных

- ID
- Автор
- Название
- ISBN

- Год издания

## Решение проблем

### Не удается добавить книгу

**Проблема:** Ошибка валидации при добавлении книги

**Решение:**
- Проверьте, что все обязательные поля заполнены
- Убедитесь, что ISBN корректен
- Проверьте диапазон года издания (1900-2100)
- Убедитесь, что количество страниц положительное

### Не удается выдать книгу

**Проблема:** Ошибка при выдаче книги

**Решение:**
- Проверьте, что экземпляр доступен (статус "Доступен")
- Убедитесь, что читатель не заблокирован
- Проверьте, что срок действия читательского билета не истек

### QR-код не распознается

**Проблема:** Сканер не распознает QR-код

**Решение:**
- Убедитесь, что QR-код четко напечатан
- Проверьте освещение
- Попробуйте ввести инвентарный номер вручную

## Приложения

### Приложение А: Форматы данных

**ISBN-10:** 10 цифр, последняя может быть X
Пример: 5-8459-0080-6

**ISBN-13:** 13 цифр, начинается с 978 или 979
Пример: 978-5-8459-0080-6

**Инвентарный номер:** Формат INV-XXXXX
Пример: INV-00001

**Номер читательского билета:** Формат RD-XXXXX
Пример: RD-00001

### Приложение Б: Рекомендации по работе

1. Регулярно создавайте резервные копии базы данных
2. Проверяйте просроченные выдачи еженедельно
3. Обновляйте данные читателей при продлении билетов
4. Используйте QR-коды для ускорения процесса выдачи/возврата
5. Регулярно проверяйте состояние инвентарных экземпляров
